#  Vue Todo App

A simple and responsive Todo List application built with Vue 3 and Tailwind CSS.  
It allows users to add, edit, delete, filter tasks, and stores data in local storage

---

##  Features

- Add new todos
- Edit existing todos
- Delete todos with confirmation
- Mark tasks as completed
- Filter tasks (All / Pending / Completed)
- Dark / Light mode toggle
- Auto save using localStorage
- Responsive UI for mobile & desktop
